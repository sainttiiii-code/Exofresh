# Exofresh 🍦

Paletas exóticas hechas con energía tropical.

## Descripción

Exofresh es una tienda de paletas artesanales que mezcla frutas, raíces, hierbas y texturas inesperadas para crear paletas frescas, coloridas y memorables.

## Sabores

- **Poroto** - Cremosa y sorprendente
- **Camote** - Dulzor natural y cálido
- **Pistacho + Piña + Yerba Buena** - Crujiente y fresco
- **Yuca** - Delicado y tropical
- **Matcha** - Elegante y herbal
- **Frutilla + Remolacha** - Frutal e intensa
- **Zanahoria + Mango** - Soleada y jugosa

## Tecnología

- **Backend**: Python + Flask
- **Frontend**: HTML + CSS + JavaScript puro
- **Deploy**: Render

## Instalación

```bash
pip install -r requirements.txt
python app.py
```

## Deploy en Render

1. Conecta este repositorio a Render
2. Usa `gunicorn app:app` como comando de inicio
3. Tu app estará disponible en `https://exofresh-xxxx.onrender.com`
